## A list of shortcuts

| Mac | Win | Description |
| :---: | :---: | --- |
| `⌘` + `Z` | `Strg` + `Z` | undo the last action (any limitation?) |
| `⇧` + `⌘` + `Z` | `⇧` + `Strg` + `Z` | redo the last action  |
